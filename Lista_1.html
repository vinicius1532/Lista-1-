    • Como funciona a arquitetura cliente x servidor?

    A arquitetura cliente servidor é uma arquitetura de aplicação distribuída, ou seja, na rede existem os fornecedores de recursos ou serviços a rede, que são chamados de servidores, e existem os requerentes dos recursos ou serviços, denominados clientes.

    O que é um protocolo? Cite exemplos

    Protocolo é o conjunto das informações, decisões, normas e regras definidas a partir de um ato oficial, como audiência, conferência ou negociação, por exemplo.

    O que é a requisição HTTP?

    Requisições HTTP são mensagens enviadas pelo cliente para iniciar uma ação no servidor. Suas linhas iniciais contêm três elementos: Um método HTTP, um verbo (como GET , PUT ou POST ) ou um nome (como HEAD ou OPTIONS ), que descrevem a ação a ser executada.

    O que é GET, HEAD e POST, cite exemplos.

    GET: O método GET é usado para solicitar dados de um servidor. Ele envia os dados como parte da URL e é amplamente utilizado para recuperar informações. Por exemplo, se você quiser obter o conteúdo de uma página da web, pode usar o método GET. Aqui está um exemplo de uma solicitação GET em HTML: HEAD: O método HEAD é semelhante ao GET, mas solicita apenas os cabeçalhos da resposta do servidor, sem o corpo da resposta. É frequentemente usado para obter informações sobre um recurso, como verificar o status da disponibilidade ou atualização de um arquivo. Aqui está um exemplo de uso do método HEAD em HTML: POST: O método POST é usado para enviar dados ao servidor como parte do corpo da solicitação. É frequentemente usado para enviar informações de formulário, como envio de formulários de login ou envio de dados para serem processados pelo servidor. Aqui está um exemplo de uso do método POST em HTML: Neste exemplo, quando o formulário é enviado, os dados do campo "username" e "password" são enviados para o servidor como parte do corpo da solicitação POST. O servidor pode então processar esses dados, por exemplo, autenticando o usuário com o nome de usuário e senha fornecidos.
